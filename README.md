# TranscribeBroadcast

A starter Node.js backend with a React frontend.

## Run locally

1. Install dependencies:

```bash
npm run install-all
```

2. Start the development environment:

```bash
npm run dev
```

3. Open React frontend:

http://localhost:3000

The React app proxies API requests to the Node backend on http://localhost:5000.

## Build for production

```bash
npm run build
npm start
```

- `GET /api/hello` - sample API endpoint
